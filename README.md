## 📌 Project Overview

This project detects **Bangladesh Banknotes** in real-time using webcam/video input.

Supported denominations:

* 2 Taka
* 5 Taka
* 10 Taka
* 20 Taka
* 50 Taka
* 100 Taka
* 200 Taka
* 500 Taka
* 1000 Taka

The model was trained using a custom dataset with synthetic background augmentation for real-life detection.
